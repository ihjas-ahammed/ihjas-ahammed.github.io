Limits & Derivatives

Derivatives of Standard Functions

  1. xn = nxn-1
  2. cosx = -sinx
  3. sinx = cosx
  4. cotx = -cosec2x
  5. tanx = sec2x
  6. secx = tanx.secx
  7. cosecx = -cotx.cosecx

Trignometric Results

    Set 1

  1. sin(A+B) = sinA.cosB + cosA.sinB
  2. sin(A-B) = sinA.cosB - cosA.sinB
  3. cos(A+B) = cosA.cosB - sinA.sinB
  4. cos(A-B) = cosA.cosB + sinA.sinB
  5. tan(A+B) = (tanA + tanB) / (1 - tanA.tanB)
  6. tan(A-B) = (tanA - tanB) / (1 + tanA.tanB)
  7. cot(A+B) = (cotA.cotB - 1) / (cotA + cotB)
  8. cot(A-B) = (cotA.cotB + 1) / (cotB - cotA)

    9. Set 2

  9. sin(A+B) + sin(A-B) = 2sinA.cosB
  10. sin(A+B) - sin(A-B) = 2cosA.sinB
  11. cos(A+B) + cos(A-B) = 2cosA.cosB
  12. cos(A+B) - cos(A-B) = -2sinA.sinB

    13. Set 3

  13. sinC + sinD = 2sin((C+D)/2).cos((C-D)/2)
  14. sinC - sinD = 2cos((C+D)/2).sin((C-D)/2)
  15. cosC + cosD = 2cos((C+D)/2).cos((C-D)/2)
  16. cosC - cosD = -2sin((C+D)/2).sin((C-D)/2)

    17. Set 4

  17. sin2A = 2sin.cosA
  18. sin2A = 2tanA/(1+tan2A)
  19. cos2A = cos2A - sin2A
  20. cos2A = 2cos2A - 1
  21. cos2A = 1 - 2sin2A
  22. cos2A = (1-tan2A)/(1+tan2A)
  23. tan2A = 2tanA/(1-tan2A)

    24. Set 5

  24. sin2A = (1-cos2A)/2
  25. cos2A = (1+cos2A)/2
  26. tan2A = (1-cos2A)/(1+cos2A)

    27. Set 6

  27. sin3A = 3sinA - 4sin3A
  28. cos3A = 4cos3A - 3cosA
  29. tan3A = (3tanA - tan3A)/(1-3tan2A)
